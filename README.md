# Work Hub — мобільний трекер щоденних задач

Простий застосунок для щоденних робочих задач, заточений під телефон. Працює
офлайн, встановлюється на домашній екран (PWA), не потребує сервера — усі дані
зберігаються локально в браузері.

## Можливості

- **Екран задач** з групами `Сьогодні / Завтра / На тижні / Потім` (як у знайомих
  таск-менеджерах) та лічильником виконаного за день.
- **Класифікація задач за трьома осями:**
  - **Операційні** vs **Розвиток бізнесу** (тип);
  - **Регулярні** vs **Разові** (повторення: щодня, кожні N днів, по днях тижня,
    щомісяця);
  - **Прості** («зробив і забув») vs **Складні** (з підзадачами / чеклістом).
- **Списки-категорії** з кольорами (Особисте, Інше, Розвиток бізнесу, Операційка,
  Регулярні) + фільтр у бічному меню.
- **Статистика / трекер:** скільки виконано за день, тиждень, місяць, рік;
  середні значення за кожен період; серія днів поспіль; діаграми за 30 днів і
  12 місяців; розбивка операційка/бізнес, регулярні/разові, прості/складні та за
  списками.
- **Цілі** з дедлайнами, кроками (milestones) і прогресом.
- **Нагадування** (час) і **нотатки** до задач.
- **Резервні копії:** експорт/імпорт усіх даних у JSON.

## Запуск

Це статичний сайт — жодного збирання не потрібно.

**Локально:**

```bash
# з кореня репозиторію
python3 -m http.server 8000
# відкрий на телефоні/у браузері: http://localhost:8000
```

Або просто відкрий `index.html` у браузері (service worker для офлайну
активується лише коли сайт відкрито через http/https).

**Розгортання (GitHub Pages):** увімкни Pages для гілки — сайт одразу доступний
за посиланням, яке можна відкрити на телефоні й «Додати на головний екран».

## Структура

```
index.html                 розмітка + підключення
assets/css/styles.css      мобільні стилі (світла/темна тема)
assets/js/store.js         дані, збереження (localStorage), бізнес-логіка
assets/js/app.js           інтерфейс: рендеринг, події, форми
manifest.webmanifest       PWA-маніфест
sw.js                      service worker (офлайн)
assets/icons/              іконки застосунку
```

Дані живуть у `localStorage` цього пристрою. Для перенесення на інший пристрій
скористайся експортом/імпортом у налаштуваннях (шестерня у бічному меню).
